# MakeYourText - 리라이트 서비스

톤(말투) + 목소리(음성 미리듣기)를 결합한 완성형 커뮤니케이션 도구

## 기능

- 텍스트 리라이트 (재작성)
- 톤 프리셋 선택 (교양/친근/단호/유머/사과/강경 등)
- 강도 슬라이더 (차분↔강함, 부드러움↔단호함)
- 음성 미리듣기 (Web Speech API)
- 안전장치 (위험 문구 차단)
- 요금제별 기능 제한

## 설치 및 실행

```bash
# 모든 패키지 설치
npm run install:all

# 개발 서버 실행 (포트 5000)
npm run dev:server

# 개발 클라이언트 실행 (포트 3333)
npm run dev:client
```

## 프로젝트 구조

- `server/` - Express 백엔드 (TypeScript)
  - `src/types/` - 타입 정의
  - `src/data/` - 프리셋 데이터
  - `src/services/` - 비즈니스 로직 (리라이트 엔진, 안전 검사, 요금제 제한)
  - `src/routes/` - API 라우트
- `client/` - React 프론트엔드 (TypeScript)
  - `src/components/` - UI 컴포넌트
  - `src/services/` - API 클라이언트, TTS 서비스

## 샘플 테스트

### 테스트 케이스 1: 교양 있게 (성인/이메일)
- **원문**: "내일까지 자료 보내 주세요."
- **톤**: 교양 있게
- **독자**: 성인
- **목적**: 요청
- **예상 결과**: 정중하고 격식 있는 문장으로 변환

### 테스트 케이스 2: 강경 (시정요구/기한 명시)
- **원문**: "내일까지 자료 보내 주세요."
- **톤**: 강경(시정요구/기한 명시)
- **독자**: 성인
- **목적**: 요청
- **예상 결과**: 기한과 후속 절차가 명확하지만 위협 표현 없음

### 테스트 케이스 3: 초등 1학년
- **원문**: "내일까지 자료 보내 주세요."
- **톤**: 친근하게
- **독자**: 초등 1학년
- **목적**: 요청
- **예상 결과**: 쉬운 단어와 짧은 문장으로 변환

## 요금제별 제한

### 무료 (Free)
- 일일 리라이트: 10회
- 주간 리라이트: 50회
- 결과 버전: 1~2개
- 보이스 프리셋: 1~2개
- 음성 재생: 20회/일
- 히스토리: 10개

### 개인 유료 (Pro)
- 일일 리라이트: 100회
- 주간 리라이트: 500회
- 결과 버전: 3개 (짧게/표준/자세히)
- 보이스 프리셋: 10개
- 음성 재생: 1000회/일
- 히스토리: 1000개

### 팀/기업 (Business)
- 일일 리라이트: 1000회
- 주간 리라이트: 5000회
- 결과 버전: 3개
- 보이스 프리셋: 20개
- 음성 재생: 10000회/일
- 히스토리: 10000개

## 안전장치

- 위험 문구 자동 차단 (폭력/해악/불법 위협 등)
- 강경 톤에서도 법적 절차 수준의 표현만 허용
- 차단 시 안전한 대체 문장 제안

## 기술 스택

- **백엔드**: Node.js, Express, TypeScript
- **프론트엔드**: React, TypeScript
- **TTS**: Web Speech API (MVP), 확장 가능한 Provider 인터페이스
